# ✈️ Intelligent Aviation Cargo Analytics (AeroLogic AI)

### Bridging Aviation Expertise with Agentic AI for Cargo Intelligence 🤖

---

## 🚀 Overview

**Intelligent Aviation Cargo Analytics (AeroLogic AI)** is an **AI-powered cargo intelligence system** designed to transform aviation logistics data into **real-time, actionable insights**.

Built on **82,600+ real cargo shipment records**, this system enables operational teams to interact with complex datasets using **natural language**, eliminating the need for manual SQL queries.

👉 This project combines **Agentic AI + Data Analytics + Aviation Domain Knowledge** to solve real-world logistics challenges.

---

## 🎯 Problem Statement

In air cargo operations:

* Massive datasets remain underutilized
* Decision-making depends on manual SQL queries
* Security risks (e.g., undeclared dangerous goods) are hard to detect
* Insights are slow and fragmented

👉 Data becomes a **“black box” instead of a decision-making tool**

---

## 💡 Solution

This project introduces an **AI-powered cargo assistant** that:

* Converts natural language into SQL queries
* Analyzes large-scale cargo data instantly
* Detects operational and security risks
* Generates dynamic visual insights
* Ensures data privacy with local LLM execution

---

## ⚡ Key Features

### 🧠 Natural Language → SQL

* Query data using plain English
* Example:
  *“Show top delayed shipments last month”*

---

### 🚨 Automated Risk Detection

* Identifies anomalies such as:

  * Undeclared Dangerous Goods (DG)
  * Suspicious shipment behavior
* Improves safety and compliance

---

### 📊 Dynamic Data Visualization

* Generates Python code automatically
* Creates charts using Matplotlib for:

  * Cargo volume trends
  * Delay analysis
  * Route performance

---

### 🔐 Hybrid Privacy Architecture

* LLM runs locally (**Gemma 3 via Ollama**)
* Sensitive aviation data remains secure
* Uses scalable cloud database (**Supabase**)

---

## 🧠 System Architecture

```text
User Query (Natural Language)
        ↓
Agentic AI Engine
        ↓
Local LLM (Gemma 3 via Ollama)
        ↓
SQL Query Generation
        ↓
Supabase Database (82K+ Records)
        ↓
Data Retrieval
        ↓
Python Code Generation (Matplotlib)
        ↓
Insights + Visualization Output
```
